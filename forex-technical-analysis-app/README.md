# Forex Technical Analysis App

A simple Streamlit app for exploring forex pairs with common technical indicators.

## Features
- Select forex symbols such as EUR/USD, GBP/USD, USD/JPY, and others
- Choose a time period and interval
- View candlestick charts with SMA, RSI, MACD, and Bollinger Bands
- See a simple bullish/bearish/neutral signal summary

## Run locally
1. Create and activate a Python environment
2. Install dependencies:
   `pip install -r requirements.txt`
3. Start the app:
   `streamlit run app.py`
